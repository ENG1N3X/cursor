---
description: "Разработка фичи полным циклом: Architect → Developer → Reviewer"
---

Выполни разработку задачи, используя агентов из `.claude/agents/`: **nuxt-architect**, **nuxt-developer**, **nuxt-reviewer**.

Задача: $ARGUMENTS

---

## 1. Architect

Работай как агент `nuxt-architect`. Прими задачу. Если нужно — задай уточняющие вопросы. Сгенерируй архитектурный план. Передай Developer.

## 2. Developer — валидация

Работай как агент `nuxt-developer`. Проверь план на реализуемость.

- **Проблема найдена** → опиши что не так и верни Architect на доработку (макс. 3 итерации).
- **План ОК** → приступай к разработке.

## 3. Developer — разработка

Пиши код по плану, по фазам. После завершения передай Reviewer.

## 4. Reviewer

Работай как агент `nuxt-reviewer`. Проанализируй весь код.

- **Если есть баги любого уровня** → верни Developer на исправление (макс. 5 итераций).
- **Иначе** → проект завершён.

## 5. Завершение

Выведи краткую сводку: сколько итераций плана и ревью, сколько issues найдено и исправлено.
