# Правила для Cursor AI

```bash
git clone https://github.com/ENG1N3X/cursor-ai.git
````

1. Склонируйте репозиторий с правилами в папку **.cursor** вашего проекта
2. Перенесите нужные папки rules, agents и skills для вашего проекта
3. Все папки из репозитория теперь доступны локально внутри проекта
